# 3d-HmIP-STHO-Screen
Stevenson screen for Homematic-IP temperature sensor HmIP-STHO 

This is a remix of "Stevenson Screen for Homematic outdoor temparature/humidity sensor" by OFreddy, that can be found here:
https://www.printables.com/model/376757-stevenson-screen-for-homematic-outdoor-temparature

Modified to have bigger mount. Consists of two parts only - the mount and the cover that slides into the mount.
Printed using ASA by a printing service, survived the first winter with a lot of snow, -25C and strong winds.
